## Police payment and yard flow

Police calls receive:

```text
paymentStatus = police_impound
payer = Police / agency impound contract
```

Each delivered target is converted to a RedFox yard record with:

- model/configuration/name/value;
- source call type;
- destination yard ID;
- `legalStatus = impound_hold`;
- configured hold duration;
- storage daily rate;
- police-hold note.

The target vehicle is then removed from the live scene. The agency reward is queued only after every target in the event is delivered.

New helper paths include:

```text
storePoliceImpoundTarget
finishPoliceImpoundEvent
```

## Emergency-scene support

A first RedFox responder layer was added through helpers that:

- search the installed vehicle catalog for police/sheriff/patrol/trooper configurations;
- search for ambulance/EMT/paramedic configurations with a vanilla ambulance fallback;
- spawn support vehicles at road-relative offsets;
- request emergency lights/hazards and parking brake;
- spawn cone props in a basic taper/checkpoint pattern;
- add all responders and cones to the event's cleanup ID list.

Support counts currently vary by call type:

```text
Accident:             2 police, 1 ambulance, 7 cones
Semi rollover:        2 police, 1 ambulance, 6 cones
Rolled-car recovery:  1 police, 0 ambulance, 5 cones
Street-race impound:  2 police, 0 ambulance, 7 cones
DUI checkpoint:       2 police, 0 ambulance, 10 cones
Police impound:       1 police, 0 ambulance, 4 cones
```

These counts and placements are test values, not final scene design.

## Optional-mod adapter state

Runtime state now includes:

```text
randomEventsDetected
randomEventsActive
trafficRebornDetected
trafficRebornRunning
trafficRebornEnforce
trafficRebornPoliceRatio
sandboxToolsDetected
```

The status is probed every five seconds and can also be refreshed from Settings.

### Random Events adapter

Uses only detected manager functions:

- `getStatusJson()` for active type/count display;
- `toggleUI()` to open or close the original panel.

No Random Events vehicle ID is taken or modified.

### Traffic Reborn adapter

Uses only detected manager functions:

- `isRunning()`;
- `spawn()` when explicitly requested and stopped;
- `setPoliceRatio(0.25)`;
- `getPoliceRatio()`;
- `setEnforce(true/false)`;
- `getEnforce()`.

New public RedFox functions:

```lua
probeOptionalMods()
enableTrafficRebornPoliceSupport()
disableTrafficRebornPoliceSupport()
```

### Sandbox Tools

Presence detection only. No call dependency.
