# JOB-09 v0.2.3 Source Change Summary

This file records the recoverable source-level change set from v0.2.1. The full runtime source remains inside the exact candidate ZIP identified in the handoff.

## Runtime file

`lua/ge/extensions/redfoxTowRecoveryDispatch.lua`

## Main changes

- Version changed from 0.2.1 to 0.2.3.
- Added road-graph candidate discovery using `map.getMap().nodes` and linked road segments.
- Added road heading calculation and longitudinal/lateral road-relative positioning.
- Replaced parking-space anchors for accident, rolled-car, and semi-rollover calls with road or roadside/shoulder anchors.
- Added vehicle classification: passenger, semi, trailer, bus, heavy, and non-road.
- Added weighted road-scene plans:
  - `two_vehicle_lane_collision`
  - `three_vehicle_chain`
  - `shoulder_secondary_collision`
  - `semi_trailer_jackknife`
  - `shoulder_semi_trailer_collision`
  - `heavy_cross_lane`
  - `bus_multi_vehicle`
- Added directional impact vectors instead of vertical-only settling impacts.
- Added adjustable `savedSceneWeight`, `eventLibrarySceneWeight`, and `roadsideSceneChance` settings.
- Default matching saved-layout priority is 0.80.
- Added scene-provider registry and contract:
  - `registerSceneProvider`
  - `unregisterSceneProvider`
  - `getSceneProviderContract`
- Added safe public-function probing of `gameplay_events_freeroamEvents`.
- Preserved all v0.2.1 routing, yards, payment, impound, history, theme, and destination-provider systems.
