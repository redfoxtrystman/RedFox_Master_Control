# JOB-09 v0.2.10 — Towing / Logistics Progression Change Summary

## Baseline

`v0.2.9 — Active Call Recovery, Alerts, and Damage`

## Main source changes

### `lua/ge/extensions/redfoxTowRecoveryDispatch.lua`

- Added progression keys:
  - `careerSkills-towing`
  - `logistics-delivery`
- Added cached category XP configuration loading.
- Expanded the shared progression builder so every completed Tow call can award:
  - Repo XP;
  - Tow & Recovery XP;
  - Logistics XP;
  - Beam XP;
  - Labourer;
  - Civil Service where applicable.
- Added category-specific bases and distance/workload/target scaling.
- Added Tow & Recovery and Logistics rewards to the Career payment payload.
- Preserved RLS `addRepossession` updates on the assigned service truck.
- Added progression class and XP breakdown to active-call UI and History.
- Added progression availability and skill levels to module status.
- Added read-only progression preview API:

```text
getProgressionPreviewForType(eventType, targetCount, responseDistance, towDistance)
```

### New skill definition

```text
gameplay/domains/careerSkills/skills/towing/info.json
```

- Display name: `Tow & Recovery`
- Attribute key: `careerSkills-towing`
- 50 levels
- RedFox-owned thresholds and milestone text
- Order 71, between RLS Logistics and Repo

### New progression configuration

```text
gameplay/redfox_tow_recovery_dispatch/progressionConfig.json
```

Contains editable category bases for:

- abandoned;
- standard tow;
- rollover/light recovery;
- heavy/semi recovery;
- accident cleanup;
- police impound;
- street-racing impound;
- DUI checkpoint impound.

## RLS compatibility decision

The exact uploaded RLS 2.6.8 source maps old vehicle-delivery/logistics aliases to `logistics-delivery`. JOB-09 awards only that unified key to prevent duplicate Logistics XP.

No protected RLS source or UI is replaced or bundled.
