# JOB-09 v0.2.9 Source Change Summary

Baseline: `v0.2.8_CareerDayAssetManager`  
Candidate: `v0.2.9_ActiveCallRecoveryAlertsDamage`  
Status: **BUILT — RUNTIME UNTESTED**

## Main Lua changes

File:

```text
lua/ge/extensions/redfoxTowRecoveryDispatch.lua
```

### Persistence

- Schema version raised to 5.
- Added alternating active-call snapshot files:
  - `settings/redfox/tow_recovery_dispatch_active_job_a.json`
  - `settings/redfox/tow_recovery_dispatch_active_job_b.json`
- Active calls save every 12 seconds and on Career save, mission exit, extension unload, and manual request.
- Added sequence/tombstone logic to prevent an old completed job from returning.
- Added resume/discard controls and exported functions:
  - `saveActiveCall()`
  - `resumeSavedActiveCall()`
  - `assignCurrentTowVehicle()`

### Tow validation

- Active call stores service vehicle world ID and RLS inventory ID where available.
- Added service-vehicle resolution and manual reassignment.
- More than 100 meters separation after target movement starts a 15-second cancellation countdown.
- Delivery requires service vehicle within 35 meters.
- Target vehicle cannot be assigned as service vehicle.

### Dispatch alerts

- Default/migrated offer lifetime: 300 seconds.
- Built-in `event:>UI>Missions>Info_Open` notification sound.
- `ScenarioFlashMessage` RedFox dispatch banner.
- Repeating reminders, default 30 seconds.
- Settings controls for acceptance and repeat times.

### Damage and scene composition

- Added per-target condition profiles.
- Added explicit ignition-off, AI disable, hazards, parking brake, tire deflation, breakgroups, and impact treatment.
- Added heavy-carrier/cargo scene option for semi recovery.
- Reduced automatic trailer use.
- Added integrated-carrier naming detection.
- Added modern police-configuration scoring/preference.

## Preserved behavior

- v0.2.8 Career-day storage clock.
- Organized Dispatch/Fleet/Storage/History UI.
- RLS progression and reward scaling.
- Personal Career inventory claim.
- Police/DUI/street-racing calls.
- Emergency-scene support.
- Multiple yards, Fleet Book, saved layouts, map hazard scan, optional-mod adapters, and history.

## Explicit exclusions

No stock Career/RLS override path was added. No RLS or Random Events source/assets were copied. Company fleet ownership and insurance remain blocked pending disposable-vehicle rollback testing.
